LaTeX export created as follows:

1) open index.html in LibreOffice
2) save as MS Office (docx) file
3) open in MS Office and save as docx
4) open in LibreOffice and export using Writer2Latex plugin (http://writer2latex.sourceforge.net)

steps 2) and 3) are necessary to maintain embedded images (otherwise, they will link to their actual path, not create a subdirectory of images)
if you don't care about non-embedded images, just do 1) and 4)
